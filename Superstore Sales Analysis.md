# 📊 Superstore Sales Analysis

## 📌 Project Overview

This project analyzes the Superstore Sales dataset using Python, Pandas, and Matplotlib.

The objective is to explore sales performance, customer behavior, product performance, regional performance, and sales trends through data analysis and visualization.

---

## 📌 Key Business Insights

- Technology generated the highest sales.
- Phones were the highest-selling sub-category.
- The West region generated the highest sales.
- California generated the highest state sales.
- New York City generated the highest city sales.
- Standard Class was the most frequently used shipping mode.
- Most orders had low sales values with a few high-value outliers.
